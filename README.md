# BionicArm

In this project I have made a Bionic Arm using microcontrollers.

V1.0.0
  Arduino MEGA 2560
  Servo Motor SG990
  
V1.1.0
  3D Printed Body
  Arduino MEGA 2560
  Servo Motor MG996R
  
V2.0.0
  3D Printed Body
  ESP8266 
  Servo Motor MG996R
